# 왁그리드

2021 우왁굳 연말공모전 16위 수상작 (팬게임 부문 - 3위)

윤농님의 왁좌의 게임에서 영감을 받아 개발한 왁타버스 로그라이크

본격적인 첫 게임 개발 프로젝트이자, 처음으로 왁굳형에게 조공한 무언가  
당시 후원도 해본적 없었는데 팬 게임 조공을 먼저 해버렸다

버그랑 컨텐츠 양 조정 실패로 인해 너무 아쉬움이 많이 남지만  
그래도 여러모로 배운 게 참 많았다
